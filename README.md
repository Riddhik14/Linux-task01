Part F: Linux Research Activity
1. What is Linux?
Linux is a free and open-source operating system used on computers, servers, mobile devices, and cybersecurity systems. It manages hardware resources and allows users to run applications efficiently.

2. Why is Linux Important in Cyber Security?
Linux is important in cybersecurity because most servers and security tools run on Linux. It provides powerful command-line tools, better system control, strong security features, and supports many penetration testing applications.

3. Difference Between Linux and Windows

Linux                                             |Windows
                                                  |
Open-source operating system                      | Proprietary operating system
Free to use                                       |	Usually requires a license
Highly customizable	                              | Less customizable
More secure and stable	                          | More vulnerable to malware
Popular in servers and cybersecurity	            | Popular on personal computers

4. What is a Linux Distribution?
A Linux Distribution (Distro) is a version of Linux that includes the Linux kernel along with software packages, utilities, and a user interface. Examples include Kali Linux, Ubuntu, Debian, Fedora, and Parrot OS.

5. Why do Ethical Hackers Prefer Linux-Based Operating Systems?
Ethical hackers prefer Linux because it is flexible, secure, lightweight, and supports a large number of cybersecurity tools. Distributions like Kali Linux come pre-installed with penetration testing and network analysis tools, making security testing easier.
